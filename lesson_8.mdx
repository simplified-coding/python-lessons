<h2 style="text-align: center;"> Dictionaries </h2>

Τα ```Dictionaries``` (= Λεξικά) είναι τύποι δεδομένων που μας επιτρέπουν μία προσεγμένη αποθήκευση δεδομένων σε μορφή που θυμίζει λεξικό / ευρετήριο. 
Μπορούμε να σκεφτούμε τα κλειδιά ως λέξεις στο λεξικό μας και τις τιμές ως τις ερμηνείες των κλειδιών.

Σύνταξη:
```python
<όνομα λεξικού> = {"κλειδί": "τιμή", "κλειδί": "τιμή", "κλειδί": "τιμή"}
```

---

<h2 style="text-align: center;"> Ιδιότητες λεξικών </h2>

1) Μπορούμε να ορίσουμε κλειδιά και τιμές σε οποιονδήποτε τύπο δεδομένου (π.χ. Αριθμός, Χαρακτήρας, κ.τ.λ.).
2) Μπορούμε να ορίσουμε τιμές ως λίστες!
3) Κλειδιά και τιμές μπορούν να είναι διαφορετικός τύπος δεδομένου (π.χ. το κλειδί να είναι ένας χαρακτήρας ανώ η τιμή να είναι ένας integer).

Μπορούμε να χρησιμοποιούμε ένα λεξικό (σε ένα ```print()``` για παράδειγμα) όπως με τις λίστες.

Παράδειγμα:
```python
my_dog = {"name": "Ermis", "age": 1}
print(my_dog)
print(my_dog["age"])
```

---

<h2 style="text-align: center;"> Επέκταση λεξικών </h2>

Μπορούμε να επεκτείνουμε λεξικά με πολλά δεδομένο για να είναι πιο ευανάγνωστα.
Αυτά τα δύο εργαλεία μαζί μας δίνουν τη δυνατότητα να επεξεργαστούμε λεξικά εισάγοντας και αλλάζοντας δεδομένα και διάφορα input και γενικότερα καινούρια δεδομένα που εισάγονται (από τον χρήστη) στο πρόγραμμα.

Σύνταξη:
```python
<όνομα λεξικού> = {
    "κλειδί": "τιμή",
    "κλειδί": "τιμή"
}
```

---

<h2 style="text-align: center;"> Αλλαγή τιμών </h2>

Επιπλέον, έχουμε τη δυνατότητα να αλλάξουμε τιμές από στοιχεία τα οποία βρίσκονται μέσα σε ένα λεξικό.

Σύνταξη:
```python
<όνομα λεξικού>[κλειδί] = <νέα τιμή κλειδιού>
```

Παράδειγμα:
```python
my_dog = {"name": "Ermis", "age": 1}
my_dog["age"] = 3
```
Στο παρόν παράδειγμα η μεταβλητή ```age``` από 1 θα αλλάξει σε 3.
