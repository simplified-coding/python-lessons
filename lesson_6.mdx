<h2 style="text-align: center;"> Functions (Συναρτήσεις) </h2>

Functions στην Python είναι μία συλλογή από εντολές που εκτελούνται όταν τις καλούμε.
Μπορούμε να κάνουμε δικά μας functions ή να χρησιμοποιήσουμε από τις προ-εγκατεστημένες στην Python.

Μερικές από τις μεταβλητές αυτές είναι:
```python
print(parameter)    -   Εκτυπώνει την τιμή μίας παραμετρου στην οθόνη.
input(parameter)    -   Επιτρέπει στον χρήστη να εισάγει μία τιμή σε μία παράμετρο.
len(parameter)      -   Επιστρέφει το μήκος της τιμής μίας παραμέτρου.
range(parameter)    -   Επιστρέφει μία σειρά από αριθμούς από το 0 μέχρι την τιμή της παραμέτρου.
int(parameter)      -   Μετατρέπει την τιμή μίας παραμέτρου σε ακέραιο αριθμό.
```

<h2 style="text-align: center;"> Δημιουργία και κλήση συνάρτησης </h2>

```python
def <όνομα συνάρτησης>():   # Δημιουργία συνάρτησης
    <εντολές>               # Εντολές συνάρτησης

<όνομα συνάρτησης>          # Κλήση συνάρτησης 
```
Παράδειγμα: 

```python
def my_function():
    

my_function()
```

<h2 style="text-align: center;"> Παράμετροι </h2>

Οι παράμετροι είναι προσωρινές μεταβλητές που χρησιμοποιούνται μόνο μέσα στην συνάρτηση.
Μπορούμε να ορίσουμε πολλές παραμέτρους χωρίζοντάς τις με κόμματα(,).

Σύνταξη συναρτήσεων με παραμέτρους:
```
def <όνομα συνάρτησης> (<παράμετρος 1>,<παράμετρος 2>, ...):            # Δημιουργία συνάρτησης με παραμέτρους
    <εντολές>

<όνομα συνάρτησης>(<τιμή παραμέτρου 1>, <τιμή παραμέτρου 2>, ...)       # Κλήση συνάρτησης
``` 

Στο παρόν παράδειγμα θα τυπωθεί στην οθόνη ``` Hello John ```.
```python
def my_function(name):
    print("Hello + name")

my_function("John")

Hello John
```

<h2 style="text-align: center;"> Global και Local μεταβλητές </h2>

### Global μεταβλητές
Οι Global μεταβλητές είναι μεταβλητές που δημιουργούνται εκτός της συνάρτησης και μπορούν να χρησιμοποιηθούν μέσα και έξω από αυτή.

### Local μεταβλητές
Οι Local μεταβλητές είναι μεταβλητές που δημιουργούνται μέσα στην συνάρτηση και μπορούν να χρησιμοποιηθούν μέσα (μόνο) στην συνάρτηση.

```python
x = "awesome"               # Global μεταβλητές

def my_function(var):       # Local μεταβλητές
    print(var) 
```

Έτσι συντάσσουμε κώδικα ώστε μία Local μεταβλητή να μετατραπεί σε Global:
```python
global <όνομα μεταβλητής>
```
