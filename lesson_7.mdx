<h2 style="text-align: center;"> Tuples </h2>

Τα ```Tuples``` (= Πλειάδες) είναι μία συλλογή δεδομένων όπου οι τιμές τους δεν μπορούν να αλλάξουν, σε αντίθεση με τις λίστες.

Σύνταξη πλειάδων:
```python
<όνομα πλειάδας> = (<1η τιμή>, <2η τιμή>, <...>, <Ν τιμή>)
```

** Προσοχή!! ** Το κόμμα ανάμεσα στα στοιχεία είναι πάντα απαραίτητο.

Στα ```tuples``` η αρίθμηση των στοιχείων γίνετε με άλλων τρόπο από αυτόν που γίνετε σε μία λίστα. 
Όταν θέλουμε να κάνουμε ```print``` ένα μέρος ενός ```tuples``` βάζουμε το στοιχείο έναρξης, άνω και κάτω τελεία (:) και το στοιχείο λήξης. Το πρώτο στοιχείο ενός ```tuple``` είναι το ```0``` και σαν στοιχείο λήξης μετά την άνω και κάτω τελεί βάζουμε το στοιχείο στο οποίο θέλουμε να σταματήσεις +1. Αυτό γιατί οποιοδήποτε νούμερο και να βάλουμε θα τυπωθεί το στοιχείο πριν από αυτό. Π.χ αν βάλουμε 0:3 θα τυπώσει το στοιχείο 0, 1 και 2.

```python
tuple = (1, 3, 5, 7)        # Ορισμός ενός tuple
print(tuple[0:2])

(1, 3)
```

Τα ```tuples``` είναι πολύ παρόμοια με τις Λίστες αλλά έχουν και κάποιες διαφορές. Επίσης είναι πολύ χρήσιμα διότι εάν και έχουν σχεδόν ίδιες ιδιότητες και λειτουργίες με τις λίστες, τα χρησιμοποιούμε όταν ξέρουμε ότι δεν θα αλλάξουμε κάτι από μία συλλογή. Αυτό μας προσφέρει απλότητα στον κώδικα και μεγαλύτερη ταχύτητα στην εκτέλεση αυτού.

Ομοιότητες με τις λίστες:
* Είναι συλλογές δεδομένων
* Υπάρχει σειρά στα αποθηκευμένα δεδομένα.
* Έχουν την ίδια αρίθμηση.

Διαφορές με τις λίστες:
* Τα δεδομένα μέσα σε ένα ```tuples``` δεν μπορούν να αλλάξουν.
* (Τα tuples) Ορίζονται μέσα σε παρενθέσεις.
* (Στα tuples) Είναι απαραίτητα τα κόμματα ανάμεσα στα δεδομένα.

<h2 style="text-align: center;"> For element </h2>

To ```for element``` χρησιμοποιείται για να εκτελούμε μία συνάρτηση όσες φορές όσα είναι και τα στοιχεία μέσα σε ένα τύπο δεδομένων (π.χ. tuples, lists, ...).

Σύνταξη:
```python
for <όνομα στοιχείου> in <όνομα τύπου δεδομένων>:

```

Παράδειγμα:
```python
list = [0, 2, 4, 6, 8 ,10, 12, 14]
for number in list:
    print(number)

0
2
4
6
8
10
12
14
```

<h2 style="text-align: center;"> If element </h2>

To ```if element``` χρησιμοποιείται για να εκτελούμε μία συνάρτηση εάν μέσα σε ένα τύπο δεδομένων (π.χ. lists, ranges), υπάρχει το στοιχείο το οποίο έχουμε ορίσει.

Σύνταξη:
```python
if <όνομα στοιχείου> in <όνομα τύπου δεδομένων>:

```

Παράδειγμα:
```python
tuple = [0, 2, 4, 6, 8 ,10, 12, 14]
if 8 in list:
    print(8)

8
```
