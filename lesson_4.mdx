<h2 style="text-align: center;"> Σύνταξη μίας λίστας </h2>

Οι λίστες είναι πιο σύνθετοι τύποι δεδομένων και περιέχουν μεταβλητές με συγκεκριμένη αρίθμηση.

Σύνταξη μίας λίστας:
```
<όνομα λίστας> = [<δεδομένο 1>, <δεδομένο 2>, <δεδομένο 3>, ...]
```

Οι λίστες μπορεί να περιέχουν χαρακτήρες, ακέραιους αριθμούς και δεκαδικούς αριθμούς.

```python
list_sting = ["Hello World"]
list_int= [1, 2, 3, 5, 7]
list_float= [6.14, 9.63, 12.75]
```

---

<h2 style="text-align: center;"> Ιδιότητες μίας λίστας </h2>

Οι μεταβλητές σε μία λίστα ονομάζονται στοιχεία λίστας και χωρίζονται μεταξύ τους με κόμματα (```,```) Η αρίθμηση των στοιχείων ξεκινάει από το 0. Μία λίστα μπορεί να λειτουργεί όπως μία μεταβλητή.

**Προσοχή!!** 
Όταν μέσα σε μία λίστα υπάρχουν χαρακτήρες δεν σημαίνει ότι δεν πρέπει να μπουν μέσα σε εισαγωγικά.

---


<h2 style="text-align: center;"> Τύπωση μίας λίστας </h2>

Έτσι τυπώνουμε μία λίστα και στοιχεία μίας λίστας:

```python
list = [1, ,2 ,3 ,5 ,7 ,11 ,13, 17, 19]
print(list)
print(list[1])
print(list[4])
print(list[7])

[1, 2, 3, 5, 7, 11, 13]
2
7
17
```

---

<h2 style="text-align: center;"> Αλλαγή των στοιχείων μίας λίστας. </h2>

Ένα στοιχείο μίας λίστας μπορεί να αλλάξει όταν γράψουμε το στοιχείο (με βάση την αρίθμησή του) που θέλουμε να αλλάξουμε, μέσα σε αγκύλες και μετά γράφουμετο στοιχείο με το οποίο θέλουμε να αντικατασταθεί.

```python
<όνομα λίστας>[θέση στοιχείου] = [<Νέο δεδομένο>]
```

Σε αυτό το παράδειγμα το 7ο στοιχείο (8ο στην πραγματικότητα), από 17 θα γίνει 19. Έτσι στην εκτύπωση δεν θα εμφανιστεί το 17 αλλά το 19 στην θέση του.

```python
list = [1, 2, 3, 5, 7, 11, 13, 17]
list[7] = 19
print(list)

[1, 2, 3, 5, 7, 11, 13, 17]
```
