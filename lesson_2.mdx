<h2 style="text-align: center;"> Ορισμός Μεταβλητής </h2>

Μία μεταβλητή μπορούμε να την ορίσουμε με διάφορους τρόπους δεδομένων.

Επίσης όταν θέλουμε να ορίσουμε μία μεταβλητή, που το όνομα της αποτελείτε από παραπάνω από μία λέξεις, τότε μεταξύ αυτών των λέξεων πρέπει να βάλουμε μία κάτω παύλα (_).

Ως μεταβλητή μπορεί να οριστεί: 

* **Strings** 
* **Integers**
* **Floats** 
* **Booleans**

``` python
hw = "Hello World!!"
print(hw)
Hello World!!

number = 7 
print(hw)
7

number = 3.14 
print(hw)
3.14

it_is_real = False #Boolean
print(hw)
False 
```

<h2 style="text-align: center;"> Αλλαγή μεταβλητής </h2>

Όταν ξαναγράφεις μία προϋπάρχουσα μεταβλητή τότε αλλάζει η τιμή της σε αυτή που όρισες.

```python
hw 
hw = "hello world"
print(hw)

hello world 
```

---

<h2 style="text-align: center;"> Πραξεις με μεταβλητές </h2>

Μπορείς να κάνεις πράξεις με τις μεταβλητές αν είναι ακέραιοι/δεκαδικοί σε print ή σε καινούριες μεταβλητές.
```python
number_1 = 15
number_2 = 3
number_3 = 4

print(number_1 + number_2)
18

print(number_1 - number_2)
12

print(number_1 * number_2)
45

print(number_1 / number_2)
5.0

print(number_2 ** number_3)
81

print(number_1 % number_3)
3

```

Επίσης μπορείτε να ορίσετε μία μεταβλητή σε ένα νούμερο που προκύπτει από μία πράξη δύο άλλων μεταβλητών.

```python
number_4 = number_2 + number_3
print(number_4)
7
```
