<h2 style="text-align: center;"> Δημιουργία βιβλιοθήκης </h2>

Οι ```Libraries``` (ή modules) είναι "πακέτα κώδικα" που δημιουργήθηκαν από άλλους προγραμματιστές, οι οποίοι τα δημιούργησαν έτσι ώστε εμείς να μπορούμε να τα ενσωματώνουμε στον κώδικά μας με ευκολία.
Ουσιαστικά εισάγουν ```functions``` που μας προσφέρουν μεγαλύτερη λειτουργικότητα και παραπάνω λειτουργείς και εργαλεία.

Έτσι εισάγουμε μία βιβλιοθήκη (=library):

```python
import <όνομα βιβλιοθήκης>
```

Ένα παράδειγμα είναι η βιβλιοθήκη ```datetime``` που μας επιτρέπει να εισάγουμε στο πρόγραμμά μας την τρέχουσα ημερομηνία και την ώρα.

```python
import datetime
```

---

<h2 style="text-align: center;"> Εισαγωγή πακέτου από μία βιβλιοθήκη. </h2>

Κάποιες φορές μπορεί να θέλουμε να εισάγουμε μόνο μία συνάρτηση από μία βιβλιοθήκη. Αυτό γιατί έτσι το πρόγραμμά μας γίνετε πιο "ελαφρύ", γρήγορο αλλά και αποτελεσματικό.

Έτσι συντάσσουμε την εισαγωγή πακέτου μεταβλητής:

```python
from <όνομα βιβλιοθήκης> import <όνομα πακέτου>
```

Σε αυτό το παράδειγμα εισάγουμε μόνο την ημερομηνία από τη βιβλιοθήκη ```datetime```:

```python
from datetime import date
```

---

<h2 style="text-align: center;"> Μετονομασία βιβλιοθήκης </h2>

Μπορεί να θέλουμε να μετονομάσουμε μία βιβλιοθήκη από ένα πακέτο μίας συνάρτησης. Αν και δεν προσφέρει άμεσα κάτι στο πρόγραμμα αυτό μπορεί να μας βοηθήσει στη σύνταξη κώδικα όπως και στην ευκολότερη αναγνώριση μίας βιβλιοθήκης.

Σύνταξη:
```python
from <όνομα βιβλιοθήκης> import <ονομα πακέτου> as <νέο όνομα>
```

Ένα παράδειγμα όπου το πακέτο ```pyplot``` εισάγεται ως ```plt```:

```python
from matplotlib import pyplot as plt
```

---

<h2 style="text-align: center;"> Χρήσιμες βιβλιοθήκες </h2>

- datetime -> Επιτρέπει τη λήψη την ώρας και της ημερομηνίας.
- random -> Επιτρέπει τη λήψη τυχαίων αριθμών και άλλων στοιχείων.
- os -> Επιτρέπει τον έλεγχο του λειτουργικού συστήματος.
