<h2 style="text-align: center;"> While </h2>

To `while` είναι ένα εργαλείο με το οποίο μπορείς να επαναλαμβάνεις μία εντολή (κομμάτι κώδικα) όσο μία προϋπόθεση ισχύει.

```python
while <προϋπόθεση>:
    <αποτέλεσμα>
```

Σε αυτό το παράδειγμα ορίζεται μία μεταβλητή (example) ίση με 0. Στο `while` όσο αυτή η μεταβλητή είναι ίση ή μικρότερη από 10 θα εκτελεί τις εντολές +1 στη μεταβλητή example και print την τιμή της μεταβλητής.

```python
example = 0
while example <= 10:
    example += 1
    print(example)

10
```

<h2 style="text-align: center;"> While True </h2>

Με το `while True` μπορούμε να κάνουμε τον κώδικα που βρίσκετε μέσα στο while να επαναλαμβάνεται για πάντα.

```python
while True:
```

<h2 style="text-align: center;"> For </h2>

To `For` είναι ένα ακόμα εργαλείο, αρκετά παρόμοιο με το ```while` στο οποίο δίνουμε μία ακολουθεί (range, tuple, list, κ.τ.λ.) Και αυτό για κάε στοιχείο το οποίο βρίσκετε μέσα σε αυτή την ακολουθία, τρέχει μία συγκεκριμένη εντολή.

```python
for <στοιχείο> in <ακολουθία>:
    <αποτέλεσμα>
```

Σε αυτό το παράδειγμα ένα `for` τυπώνει μία λίστα.

```python
list = [1, 2, 3, 4]
for number in list:
    print(number)

1
2
3
4
```

<h2 style="text-align: center;"> Break </h2>

Με το `break` μπορείς να σταματήσεις ένα ``while` ή ένα `for` από το να συνεχίσει να ελέγχει το εάν οι επόμενες προϋποθέσεις ισχύουν.

Σε αυτό το παράδειγμα το `break` ενεργοποιείται μετά από μία επανάληψη του κώδικα διότι η μεταβλητή `boolean` από False έγινε True και για αυτό θα τυπωθεί στην οθόνη μόνο `1`.

```python
example = 0
boolean = False

while example < 10:
    if boolean == True:
        break
    example += 1
    print(example)
    boolean = True

1
```

<h2 style="text-align: center;"> Input </h2>

Το `input` επιτρέπει στον χρήστη να εισάγει δεδομένα στο πρόγραμμα και έτσι να το κάνουμε πιο διαδραστικό κάνοντας ερωτήσεις (π.χ. Quiz) και όχι μόνο (π.χ. εισαγωγή user name).

```python
input("κείμενο")
```

Σε αυτό το παράδειγμα ο χρήστης εισάγει το όνομά του και αυτό τυπώνετε στην οθόνη του. (Δεν έχουμε βάλει το τι θα τυπωθεί στην οθόνη διότι αυτό εξαρτάτε από το τι θα εισάγει ο χρήστης.)

```python
name = input("Ποιο είναι το όνομά σου;")
print(name)

Εξαρτάται από το τι εισάγει ο χρήστης.
```
